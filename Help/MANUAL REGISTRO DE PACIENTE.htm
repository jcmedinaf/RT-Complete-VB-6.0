REGISTRO DE PACIENTE

 

El formulario Pacientes es usado para agregar, modificar, visualizar o borrar Registros del Paciente. Este formulario es utilizado para registrar todos los datos del paciente al sistema, ingresando así su numero de cedula, nombres y apellidos, fecha de nacimiento, dirección, teléfono de habitación y/o celular, sexo, ocupación, Medico tratante, medico remitente, entre otros datos que son solicitados. Figura Nº1.

 

 

 

Figura Nº 1

 

El formulario de Pacientes consta de una barra de botones de acción ubicada en la parte inferior del mismo, conformada por los botones de Agregar, Guardar, Borrar, Buscar, Deshacer, Cerrar, Flechas de desplazamientos. Figura Nº 2.

 

 

 

Figura Nº 2.

 

 

A continuación se describen el funcionamiento de cada uno de los botones mencionados anteriormente:

 

Permitirá al usuario ingresar un nuevo paciente con todos los datos que se requiere.

Permitirá guardar o actualizar la información del paciente en el sistema.

Usado para Borrar un Registro o Registro seleccionado, el cual solo podrá se borrado por el administrador del sistema o supe usuario.

Este icono es el de buscar al accionar este botón con hacer un simple clic o presionar la tecla el botón comenzara la búsqueda.

Los botones siguientes son de desplazamiento  (Derecha e Izquierda) que le permitirá ir y ver a cada paciente de uno en uno

Permite Deshacer toda la operación realizada.

El icono de salida le permitirá al usuario salir y regresar a la pantalla principal sin salir de sistema en si.

 

Otros Botones que se encuentran presenten en el sistema se describen a continuación:

Permite generar el Número de Historia Medica a cada paciente al momento de comenzar el tratamiento.

Permite tomarle una foto al paciente para se agregada al registro medico del mismo.

Muestra una lista de todos los pacientes que se encuentran registrados en el sistema.

Permite anexas o ingresar o registrar un nuevo medico.
